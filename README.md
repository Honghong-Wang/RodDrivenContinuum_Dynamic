# Rod-Length-Actuated Continuum Robot Dynamic Executability Dataset

This data-only package supports the manuscript "Reduced-Order Dynamic Executability Optimization for Rod-Length-Actuated Continuum Robots".

Author: HH  
Affiliation: University College London (UCL)

The package contains simulation-only numerical outputs, processed figures, and supplementary videos. It is organized as a clean GitHub data/release package for review, inspection, and archival sharing.

## Contents

- `supplementary_videos/`: nine MP4 videos plus the video manifest and video README.
- `simulation_data/`: CSV/MAT numerical outputs, processed figures, supplementary figures, and traceability notes used by the manuscript.
- `data_manifest.csv`: file list with size and SHA-256 hash.
- `checksums_sha256.csv`: SHA-256 checksums for integrity verification.

## Scope

All results are simulation-only. The friction and contact outputs are reduced-order numerical extensions and should not be interpreted as experimentally identified physical parameters.
